In order to build this project, you need to download the OpenCV 2.4.11 SDK for iOS found at:

  http://opencv.org/downloads.html

And install:

  opencv2.framework

in this directory.  You will also need to download the Form SDK found at:

  http://www.relativewave.com/form/sdk/

And install:

  Former.framework
  Performer.framework
  RelativeInterface.framework
  RelativeInterfaceMac.framework

in this directory.
